
#  Análisis de Inteligencia de Negocios: Panadería "Miga de Oro"

##  Descripción del Proyecto
Este proyecto consiste en el desarrollo de un ecosistema de análisis de datos en **Power BI** para la panadería coreana **Miga de Oro**. El objetivo principal fue transformar datos transaccionales brutos en un tablero de control estratégico que permitiera optimizar la toma de decisiones basada en el comportamiento del consumidor y patrones de venta temporales.

##  Hallazgos Clave (Business Insights)

* **Liderazgo de Producto:** El **Angbutter** es el motor financiero del negocio, generando el **34.54%** de la facturación total ($11,656.69).
* **Identificación de Anomalías:** Se detectó una caída crítica de ventas los **martes** ($38.21) frente al promedio diario real de **$135.54**, sugiriendo una oportunidad de investigación operativa.
* **Análisis de la "Hora Dorada":** El bloque horario de **11:00 a 14:00 (Almuerzo)** es el más lucrativo, acumulando más de **$24,000** en ventas.
* **Eficiencia de Transacción:** Se estableció un **Ticket Promedio de $14.31**, con una media de **4.6 unidades** adquiridas por cliente.
* **Balance de Categorías:** Se identificó una oportunidad de crecimiento en la categoría de **Bebidas (11.67%)**, la cual está significativamente por debajo de **Panadería (88.33%)**.
* **Precisión Financiera:** Mediante el uso de DAX, se calculó la **Venta Promedio Diaria Real ($135.54)**, eliminando sesgos de días sin registros para proyecciones fiables.

##  Recomendaciones Estratégicas

Basado en el análisis de datos, se proponen las siguientes acciones para mejorar la rentabilidad:

1. **Impulsar Ventas Cruzadas (Cross-selling):** Dado que el 88.33% de las ventas son de panadería, se recomienda implementar combos de "Producto Estrella + Bebida" (ej. Angbutter + Americano) para elevar la participación de la categoría de Bebidas.
2. **Optimización de Inventario por Franja Horaria:** Reforzar la producción de los Top 5 productos antes de las 11:00 AM para maximizar la captura de valor durante la "Hora Dorada" (11:00-14:00).
3. **Estrategia para Días de Baja Afluencia:** Implementar promociones agresivas o programas de lealtad específicos para los días martes, buscando mitigar la caída atípica de ingresos detectada.
4. **Fomento del Ticket Promedio:** Crear paquetes de "Media Docena" o "Productos Familiares" para incentivar que el promedio de 4.6 unidades por ticket suba a 6 o más unidades.

##  Stack Tecnológico
* **Power BI Desktop:** Modelado de datos y visualización avanzada.
* **DAX (Data Analysis Expressions):** Creación de medidas personalizadas para KPIs y análisis temporal.
* **Power Query:** Limpieza, normalización y transformación de datos.

##  Estructura del Reporte
1. **Resumen Ejecutivo:** Vista macro de ventas totales, unidades y distribución por categoría.
2. **Análisis de Producto:** Evaluación detallada de rentabilidad y participación de mercado de productos estrella.
3. **Análisis Temporal:** Mapas de calor por bloque horario y día de la semana para optimización operativa.
